# AI Assistants Proxy

Корпоративный сервис проксирования доступа к западным AI API для инженерной команды Организации. Обеспечивает безопасный, анонимный и отказоустойчивый доступ к AI-сервисам через единый статичный эндпоинт на территории РФ.

## Структура проекта

| Директория | Описание |
|------------|----------|
| [Clients/](Clients/) | Конфигурации клиентов (VSCode, FoxyProxy, PowerShell) |
| [Servers/](Servers/) | Серверная инфраструктура (Ansible deployment, примеры конфигураций) |
| [docs/](docs/) | Документация проекта |

## Быстрый старт

1. **Развёртывание серверов** — [Servers/deploy/README.md](Servers/deploy/README.md)
2. **Настройка клиентов** — [Clients/README.md](Clients/README.md)

## Архитектура

Двухуровневая архитектура проксирования:

- **Access tier (РФ)** — аутентификация, авторизация, журналирование, кеширование, VRRP failover
- **Upstream tier (вне РФ)** — анонимный транспорт, минимальная конфигурация, ACL по IP

Подробнее: [docs/AA/](docs/AA/) — архитектурная документация (модель C4).

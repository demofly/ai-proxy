# DD — Реестр архитектуры данных

## Описание

Архитектура данных проекта **AI Assistants Proxy** описывает информационные активы IaC-системы: конфигурационные параметры, учётные данные, списки управления доступом, списки доменов, записи журналов и секреты. Проект не использует традиционные СУБД — все данные хранятся в конфигурационных файлах, шаблонах Ansible и файлах на целевых серверах.
